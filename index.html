What Mirror Trip Does

A calm structure for travel details that usually end up scattered.

The app is designed for creating and managing trips from a mobile device, with support for itinerary details, notes, dates, destinations, activities, costs, and traveller information.

Organise full itineraries

Build trips with destinations, dates, days, activities, notes, and useful context in one place.

Capture real trip details

Track travellers, costs, and optional photo uploads alongside the trip itself.

Use location features intentionally

Current location is only used when you actively choose it for place-related trip entries.

Control visibility

Trips begin private by default, with optional publishing when you want selected content visible in the app.
